## Typography

Set in **Sentient** (Noopur Choksi & Barbara Bigosińska, [Fontshare](https://www.fontshare.com/fonts/sentient)) for the serif and display, **Switzer** ([Fontshare](https://www.fontshare.com/fonts/switzer)) for the sans, and **JetBrains Mono** ([JetBrains](https://www.jetbrains.com/lp/mono/)) for the mono. Both Fontshare faces are by [Indian Type Foundry](https://www.indiantypefoundry.com) and free for commercial use.

## Color

A warm cream paper, warm charcoal ink, and a single accent in vermillion (`#c14a2a`). Dark mode is a derivative: warm near-black surface with bone-warm text and a slightly desaturated vermillion. Light is the canonical mode; dark is opt-in via the toggle in the header.

## Stack

- **[SvelteKit 2](https://svelte.dev/docs/kit)** with **Svelte 5** runes, TypeScript
- **[Tailwind CSS 4](https://tailwindcss.com)** for layout and tokens
- **[Sanity](https://www.sanity.io)** for photography &mdash; the only content that lives outside git
- **Markdown in git** for writing, notes, projects, reading, and these small pages
- **[Cloudflare Workers](https://developers.cloudflare.com/workers/)** for hosting; **Cloudflare R2** archives photography originals
- **[oxlint](https://oxc.rs) + [oxfmt](https://oxc.rs)** for linting and formatting
- **[fnm](https://github.com/Schniz/fnm)** for Node version management; **[just](https://just.systems)** for tasks
